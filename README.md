# K-MIMS (국방병무정보시스템)

K-MIMS(Korea Defense Military Information Management System)는 대한민국의 공정하고 투명한 병역 판정 업무를 지원하기 위한 디지털 EMR 시스템입니다. 

현장의 검사관들이 수검자의 상태를 직관적으로 파악하고, 최신 법령 및 판례(RAG)를 바탕으로 신속하고 정확한 의사결정을 내릴 수 있도록 돕는 다양한 편의 기능을 제공합니다.

## ✨ 주요 기능

- **실시간 수검자 관리**: 대기 중인 수검자 현황을 한눈에 파악하고 신속하게 검사를 진행할 수 있습니다.
- **통합 병무 진단 기록**: 주관적 소견과 객관적 검사 결과(SOAP)를 한 화면에서 체계적으로 기록합니다.
- **디지털 펜 차트**: 신체 주요 부위에 대한 정밀한 표기와 메모가 가능한 그래픽 도구를 지원합니다.
- **AI 판정 보조 (RAG)**: 복잡한 병역법 규정과 과거 판례 데이터를 실시간으로 참조하여 판정 등급을 제안합니다.
- **맞춤형 작업 화면**: 사용자의 업무 환경에 맞춰 화면 레이아웃의 크기를 자유롭게 조절할 수 있습니다.

## 🚀 설치 및 실행 방법

본 프로젝트는 최신 파이썬 환경에서 동작하며, 간단한 명령어로 실행할 수 있습니다.

### 1. 프로그램 준비
아래 명령어를 입력하여 필요한 라이브러리를 설치합니다.
```bash
uv sync
```

### 2. 프로그램 실행
서버를 실행한 후 브라우저에서 제공되는 주소로 접속합니다.
```bash
uv run server.py
```
- 접속 주소: `http://localhost:8000`
